# Genshin-manger
**Genshin Impact descktop help manager** -- десктопное приложение, которое совмещает в себе функционал мобильного приложения, сайта HoyoLab (Тут ссылка) и вики по игре Genshin Impact.

# Функционал программы
**Главные функции:**
- [ ] Realtime смола
- [ ] Статистика молитв
