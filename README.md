# Функционал программы
**Главные функции:**
- [ ] Realtime смола
- [ ] Статистика молитв
- [ ] Статистика по смоле и примагемам
- [ ] Автоматическая отметка на сайте при запуске программы
- [ ] Ввод промокодов

**Второтепенные функции:**
- [ ] Сохраниние настроек и старых молитв
- [ ] Второй язык
    - [ ] Русский язык
	- [ ] Английский язык
- [ ] Автозапуск и сворачивание в трей
- [ ] Аккаунты
    - [ ] Мультиаккаунт
	- [ ] Смена сервера
